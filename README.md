# Final UrjaAI Project

## Project Description
Final UrjaAI is an innovative project aimed at revolutionizing the energy sector with Artificial Intelligence. By leveraging state-of-the-art machine learning algorithms, it aims to optimize energy consumption, improve efficiency, and provide actionable insights into energy utilization patterns.

## Features
- **Smart Energy Management**: Real-time monitoring and optimization of energy usage across various sectors.
- **Predictive Analytics**: Utilize historical data to forecast energy demand and supply, aiding in better resource allocation.
- **User-Friendly Interface**: An intuitive dashboard that allows users to visualize their energy consumption and obtain valuable insights.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/avinas29/final_UrjaAI.git
   ```
2. Navigate to the project directory:
   ```bash
   cd final_UrjaAI
   ```
3. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
- To run the application, execute:
  ```bash
  python app.py
  ```
- Access the dashboard at `http://localhost:5000`.
